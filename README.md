## Setup

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content in `index.html` with your information
4. Update the contact links and project information
5. Replace placeholder project images and links

## Customization

### Personal Information
- Edit the hero section in `index.html` to include your name and description
- Update the about section with your hardware/embedded background and statistics
- Modify project cards with your actual hardware and embedded projects

### Styling
- Colors can be adjusted in the CSS custom properties at the top of `styles.css`
- Fonts can be changed by updating the Google Fonts import and CSS font-family declarations
- Animations can be modified in `script.js`

### Content
- Add your actual hardware and embedded project links and descriptions
- Update social media links in the contact section
- Replace email address with your actual email
- Include links to PCB designs, schematics, and firmware repositories
